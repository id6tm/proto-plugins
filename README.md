# proto plugins

A collection of community [TOML plugins](https://moonrepo.dev/docs/proto/non-wasm-plugin) for [proto](https://moonrepo.dev/proto), the pluggable multi-language toolchain manager.

Maintained by [Id6](https://github.com/id6tm).

## Plugins

| Tool               | Description                        |
| ------------------ | ---------------------------------- |
| [Aspire](./aspire) | Installs the Microsoft Aspire CLI. |

See each plugin directory for installation and tool-specific usage.
